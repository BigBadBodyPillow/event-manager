# Event Manager

### Installation

> You will need [Node.js](https://nodejs.org/en) for this to run!

```
git clone https://github.com/BigBadBodyPillow/event-manager.git
```

```cmd
cd event-manager
```

```cmd
npm install
```

```cmd
npm run dev
```

Then open the address prodvided in the terminal for example http://localhost:5174/
